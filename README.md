# Pixabay 素材搜索技能

根据用户需求生成 Pixabay 搜索链接，支持 8 种素材类型。

## 支持的素材类型

| 类型 | 说明 |
|------|------|
| 音乐 | 背景音乐、配乐、BGM |
| 音效 | 环境音、转场音、拟音、冲击音 |
| 图片 | 照片，支持真实拍摄和 AI 生成 |
| 插画 | 插画作品 |
| 矢量图 | SVG 矢量素材 |
| 视频 | 视频素材、实拍、动画 |
| GIF | 动图素材 |
| 3D模型 | 三维模型文件 |

## 工作方式

1. 用户描述素材需求
2. 技能判断素材类型
3. 读取对应参考文档
4. 按"由宽到窄"原则生成 2-3 个梯度搜索链接
5. 用户点击链接浏览素材

## 安装

将 `1_Script/search-pixabay/` 目录复制到 Claude Code 的 skills 目录：

```bash
cp -r 1_Script/search-pixabay ~/.claude/skills/
```
