# 当前项目简介

Pixabay 素材搜索技能。根据用户需求生成 Pixabay 搜索链接，支持音乐、音效、图片、插画、矢量图、视频、GIF、3D模型共 8 种素材类型。

# Git 服务器

- 平台：GitHub（https://github.com/Tonyhzk）
- 认证方式：SSH（已配置 SSH Key）
- 默认分支：main

# 通用习惯

- 保持对每次小改动进行 commit 的习惯
- 小更新也要更新 VERSION 文件中的版本号
- 每次 commit 时检查是否需要同步更新 README.md 和 CHANGELOG.md
- README 遵循 `program-github` 技能中的 README 写作规范
- CHANGELOG 遵循 `program-github` 技能中的 CHANGELOG 写作规范
- LICENSE 使用 Apache 2.0，模板见 `program-github` 技能

# 本项目习惯
